================================================================================
UC Case Registry (潰瘍性大腸炎 症例登録アプリ)
================================================================================

潰瘍性大腸炎（UC）の臨床データ収集（特にJAK阻害薬とS1P受容体調節薬の比較）を
目的としたウェブアプリケーションです。

Version 2 では、入力漏れを防ぐための完全性チェック機能を追加しています。
各項目は「入力済み」または「未入力確認チェック済み」のどちらかになるまで、
正式登録時に警告されます。

[英語版は readme_en.txt をご覧ください。]

--------------------------------------------------------------------------------
主な機能
--------------------------------------------------------------------------------

* サーバー不要
  ウェブブラウザ上のみで完全に動作します。バックエンドサーバーやデータベースの
  構築は必要ありません。

* ローカルファイル保存
  モダンな File System Access API を使用し、パソコン上のローカル .json ファイルに
  直接データを保存・読み込みします（未対応ブラウザでは localStorage にフォール
  バックします）。

* バイリンガル音声入力
  音声コマンドを使用して検査値データを素早く入力できます。英語と日本語の両方の
  医療用語（例：「CRP 5.2」「白血球 6500」など）に対応しています。

* 自動計算機能
  入力されたデータに基づいて、年齢、BMI、罹病期間、PRO-2スコア、好中球実数（ANC）、
  リンパ球実数（ALC）を自動的に計算します。

* データエクスポート
  収集した症例データをCSVファイルとしてエクスポートしたり、Excelに直接貼り付け
  られる形式でコピーしたりできます。

* テーマ対応
  医療向けにデザインされたダークテーマとライトテーマを搭載し、ボタンで切り替え
  可能です。

* 未入力確認
  空欄の項目には「Confirm missing」をチェックし、意図的な未入力であることを
  確認できます。

* 登録前の完全性チェック
  Register Case を押すと、未入力かつ未確認の項目を検出します。ステップごとの
  件数バッジと上部の Data Quality パネルから該当項目に移動できます。

* 症例一覧の品質ステータス
  Case List に Quality 列を追加し、各症例の未解決チェック数を表示します。

* 英語/日本語表示切り替え
  画面右上の「日本語 / English」ボタンで、固定UI、ステップ名、フィールド名、
  選択肢、品質チェック表示を切り替えられます。保存されるコード値や
  Excel/CSV出力用の値は変更されません。

--------------------------------------------------------------------------------
使い方
--------------------------------------------------------------------------------

1. アプリを開く
   モダンなウェブブラウザで index.html を開きます（ファイル保存や音声入力の
   全機能を利用するため、Google Chrome または Microsoft Edge を推奨します）。

2. 保存先ファイルをリンクする
   - 上部ナビゲーションバーの「Link File」（フォルダアイコン）ボタンをクリック
     します。
   - パソコン上の任意の場所に cases.json ファイルを保存（または選択）します。
   - 既存のJSONファイルを選択した場合はそのデータを読み込みます。新規作成した
     場合は現在のデータが書き込まれます。
   - 注意: セキュリティ上の理由から、アプリを開くたび、またはページを更新する
     たびにファイルをリンクし直す必要があります。

3. 症例を登録する
   - 「+ New Case」タブでフォームに入力します。
   - 検査値データの横にあるマイクボタンをクリックすると音声入力が可能です。
     言語は英語（EN）と日本語（JA）で切り替えられます。
   - フォームの最後にある「Register Case」をクリックして保存します。

4. 症例の管理
   - 「Case List」タブに切り替えると、登録された症例の閲覧、検索、編集、削除が
     できます。

5. データのエクスポート
   - 「Export CSV」でデータをダウンロードします。
   - 「Copy for Excel」でデータをコピーし、Excelのシートに直接貼り付けること
     ができます。

--------------------------------------------------------------------------------
ファイル構成
--------------------------------------------------------------------------------

* index.html    : メインのユーザーインターフェース
* style.css     : デザインとテーマを含むスタイルシート
* app.js        : アプリケーションの主要なロジック
* fields.js     : フォームのステップと入力項目を定義する設定ファイル
* readme_en.txt : 英語のドキュメント
* readme_ja.txt : 日本語のドキュメント

--------------------------------------------------------------------------------
動作環境
--------------------------------------------------------------------------------

* モダンなウェブブラウザ
* 以下の機能を利用するため、Google Chrome または Microsoft Edge を強く推奨します：
  - File System Access API: ローカルファイルへの直接保存機能
  - Web Speech API: 音声入力機能
